Find That Film is a web application I developed as part of a MEAN stack course curriculum. This was an exercise in utilizing a front end framework in connection with a persistent database.Â 

TMDB (The Movie Database) provides an excellent free API for querying film information. I implemented functionality such as user register and login, film search, and the ability for users to save a list of their favorite films. This project was a fairly comprehensive introduction to Angular Framework and working in a non-relational database.

The back end is no longer hosted, but the front end is serving up on Netlify.
